# HR Attrition Analytics Dashboard (Excel)

## Project Overview
This project analyzes employee attrition using an HR dataset. The goal is to identify patterns and factors that contribute to employee turnover.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design

## Dataset
The dataset contains employee-related information such as:
- Age
- Department
- Job Role
- Monthly Income
- Years at Company
- Job Satisfaction
- Attrition Status

## Data Cleaning
The following steps were performed:
- Handled missing values
- Removed duplicate records
- Standardized column names
- Corrected data formats

## Dashboard Features
The dashboard includes:

- Attrition Rate KPI
- Attrition by Department
- Attrition by Age Group
- Attrition by Job Role
- Years at Company Analysis
- Interactive filters

## Insights
Some key insights derived from the analysis:

- Higher attrition observed in certain departments
- Employees with fewer years at company show higher turnover
- Job satisfaction impacts attrition trends

## How to Use
Download the Excel file and interact with the dashboard filters to explore employee attrition trends.
